# M1986: INDX purge-bucket pellet counter

`M1986` manages the INDX nozzle-cleaner purge bucket. With no parameters it
parks the tool, asks the user to empty the bucket, and resets the counter after
confirmation.

```gcode
M1986
```

The counter and automatic-pause settings can also be managed by a slicer or
serial host:

```gcode
M1986 Q       ; report current pellets, pause threshold and physical capacity
M1986 S1200   ; signal/set the current count to 1200 pellets
M1986 P2500   ; pause to empty the bucket at 2500 pellets
M1986 P0      ; use the installed bucket's physical capacity as the threshold
M1986 R       ; reset the count without moving or displaying the empty prompt
```

`Q`, `S`, `P`, and `R` return one machine-readable line:

```text
PURGE_BUCKET pellets=1200 threshold=2500 capacity=3000
```

The threshold is also available on the printer under **Nozzle Cleaner > Pause
At Pellets**. `Capacity` tracks the selected normal or extended bucket size.
Firmware clamps a configured threshold to the installed bucket's physical
capacity.

When the threshold is crossed during a print and **Pause on Full Nozzle
Cleaner** is enabled, the firmware parks and waits for the bucket to be
emptied. During a serial print it emits `pause`/`paused purge_bucket_full` host
actions, followed by matching resume actions only after the prompt is cleared.
