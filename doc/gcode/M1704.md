# M1704 — MMU filament load test

`M1704` runs the MMU path load test without loading filament into the nozzle.
The printer UI exposes the same operation under **Control > Loading test**, with
one entry per enabled slot and a **Test All** entry.

```gcode
M1704 P2 M0 ; test MMU slot 3 (zero-based slot 2)
M1704 K21 M0 ; test slots 1, 3, and 5 (bit mask 0b10101)
M1704 A M0 ; test every enabled MMU slot
```

`P` selects one slot. `K` supplies a decimal bit mask, and `A` selects all
enabled slots. `M0` means the indices are physical MMU slots; omit it only when
the supplied index should pass through active tool mapping.

The selected tests run sequentially and stop on the normal MMU error or user
abort path. `A` and `K` ignore disabled virtual slots. For a five-slot MMU,
`K21` is `10101` in binary and therefore selects physical slots 0, 2, and 4.
